# BearGrub

Full-stack project that surfaces halal, vegetarian, and vegan options in UC Berkeley dining halls. A Flask API scrapes Cal Dining’s daily XML menu feeds, uses OpenAI to filter halal-friendly dishes, and serves meal data to a Next.js frontend in `halaliverse-app`.

## Repository Layout
- `app.py` — Flask API exposing `/api/halal-meals`, `/api/vegan-meals`, `/api/vegetarian-meals`.
- `MealClassification.py`, `generate_data.py` — scripts for fetching menus and classifying meals (writes `halal.json`).
- `gpt_classifier.py` — OpenAI helper that flags non-halal meat ingredients.
- `halal.json` — cached halal output (regenerated by `generate_data.py`).
- `halaliverse-app/` — Next.js 14 UI that lets users pick meal type and mealtime and view dishes per dining hall.
- `Archive/`, `TestJson/` — older experiments and sample data.

## Prerequisites
- Python 3.9+ with `pip`.
- Node.js 18+ and `npm`.
- OpenAI API key (set `OPENAI_API_KEY` in your environment).

## Backend: Flask API
1) Install dependencies:
```bash
python -m venv .venv
source .venv/bin/activate
pip install flask flask-cors requests openai
```
2) Configure the key (example for macOS/Linux):
```bash
export OPENAI_API_KEY="sk-..."
```
3) Run the server:
```bash
python app.py
```
The API listens on `http://127.0.0.1:5000`.

### Available Endpoints
- `GET /api/halal-meals`
- `GET /api/vegan-meals`
- `GET /api/vegetarian-meals`

Each endpoint returns a JSON object keyed by dining hall (`Cafe_3`, `Crossroads`, `Foothill`, `Clark_Kerr_Campus`) with `Breakfast`, `Brunch`, `Lunch`, and `Dinner` arrays of dish names.

### How Halal Detection Works
1. Pull daily XML menus from `https://dining.berkeley.edu/wp-content/uploads/menus-exportimport/{DINING_HALL}_{YYYYMMDD}.xml`.
2. Filter out items containing alcohol or pork allergens.
3. Send remaining ingredient lists to GPT (see `gpt_classifier.py`) to exclude non-halal meats unless explicitly marked HALAL/KOSHER.

## Frontend: Next.js App (`halaliverse-app`)
1) Install dependencies:
```bash
cd halaliverse-app
npm install
```
2) Start the dev server:
```bash
npm run dev
```
Visit `http://localhost:3000`. The UI fetches from the Flask API at `http://127.0.0.1:5000`, so keep the backend running.

## Regenerating Cached Data
To write a fresh `halal.json` snapshot (uses the OpenAI key):
```bash
python generate_data.py
```

## Notes and Limitations
- The OpenAI API key must be provided; `gpt_classifier.py` contains a placeholder.
- Meal availability depends on Cal Dining publishing the daily XML feed for the current date.
- No automated tests are included yet.
